# Sistema de Cadastro

## Descrição

Aplicação de CRUD para gerenciamento de cadastro desenvolvida com Angular 15 e persistência em localStorage.

## Instalação

**Pré-requisitos:** Node.js v16+ e npm v8+

```bash
cd crud
npm install
```

## Como Usar

**Desenvolvimento:**
```bash
npm start
```
Acesse `http://localhost:4200/`

**Build de Produção:**
```bash
npm run build
```

**Testes:**
```bash
npm test
```

**Modo Watch:**
```bash
npm run watch
```
